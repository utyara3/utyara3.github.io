## Что это такое?

Для начала разберемся с обозначениями:

AAC - Attack Animation Cancel  
ПАА - Пропуск Анимации Атаки

ПАА - это пропуск perf состояния оружия для того чтобы тратить меньше кадров на атаку, и, соответственно, бить быстрее.

Подробнее про состояния оружия можно почитать в гайд разделе.

Сделать это можно для любого оружия, у которого есть это perf состояние, просто реэкипировав это оружие во время этого состояния.  
Вручную это сделать очень сложно и бессмысленно, поэтому обратимся к коду.

Первоначальная версия ААС была очень громосткой: это был большой код, который теоретически работал, но был перегружен. Путем развития, люди в дискорде пришли к этому:

```
?item.right.state = 3
  equipR *0 +0 shield
  equip @item.right@
?item.left.state = 3
  equipL *0 +0 sword
  equipL @item.left@
```

Это отличный код, который можно просто скопировать и вставить его в самый верх камня (главное чтобы он был перед всеми экипировками).  

Но, и этот код не идеален. Он работает некорректно, если оружие имеет 30 апс. Это может быть, к примеру, посох +11, зачарованный на скорость атаки, при использовании зелья берсерка. Тогда, код нужно улучшить:
```
?item.right.state = 3
  ?totaltime % 2 = 0
    equipR *0 +0 shield
  :
    equipR stone
  equip @item.right@
?item.left.state = 3
  equipL *0 +0 sword
  equipL @item.left@
```

Также, можно избавиться от экипирования "мусорных" оружий, которые используются лишь для реэкипировки оружия.  

Вместо них можно использовать те оружия, которые вы всегда используете на локации.  

Это может быть, к примеру, трискелион, дэш щит, камень звезды и т.п., но это уже мелочи, которые можно отнести к спидраннерским, т.к. на низких звездах не хватает гп (очков мощи) и приходится изхищряться разными способами:
```
?item.right.state = 3
  ?totaltime % 2 = 0
    equipR dash +0 *0 shiny
  :
    equipR trisk
  equip @item.right@
?item.left.state = 3
  equipL trisk
  equipL @item.left@
```

Подведя итоги, скажу, что для обычного игрока с головой хватит и первого варианта, и смысла использовать остальные нет, разве что только второй, но только если вы уже на продвинутом этапе игры.
